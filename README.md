# Web 计算器 / Web Calculator

一个使用原生 HTML、CSS、JavaScript 编写的**科学风格 Web 计算器**，支持括号、幂、平方根、取余和历史记录、主题切换等功能。

## 功能特性

- **基础运算**：加、减、乘、除、百分号 `%`
- **高级运算**：幂运算 `^`、平方根 `√`
- **表达式解析**：支持括号与一元负号（如 `-3^2`）
- **历史记录**：保留最近 10 条计算，点击可重新计算
- **键盘输入**：支持数字键、运算符、Enter、Backspace、Esc 等
- **主题切换**：浅色 / 深色 / 跟随系统

## 目录结构

- `index.html`：页面结构，引用外部样式与脚本
- `style.css`：页面样式（布局、配色、动画等）
- `main.js`：计算逻辑与交互脚本
- `.gitignore`：Git 忽略规则

## 本地运行

1. 克隆或下载本项目到本地。
2. 直接用浏览器打开 `index.html` 即可使用（不依赖任何构建工具或后端服务）。

## 开发说明

- 修改样式：在 `style.css` 中编辑对应的类（如 `.calculator-shell`、`.key` 等）。
- 修改逻辑或新增功能键：在 `index.html` 中添加按键，并在 `main.js` 中扩展处理逻辑。
